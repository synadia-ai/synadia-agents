## [0.1.0] - 2026-06-22

### Added

- Initial Codex app-server-backed Synadia Agent Protocol adapter package.
- Managed, attached, and opt-in manager modes for exposing Codex sessions over NATS.
- Permission approval/denial mapping for Codex app-server command, file-change, and permission-grant requests.
- Protocol, fake-runtime, session-manager, future-watch, permission, and live approval validation harnesses.
- README setup, discovery, validation, troubleshooting, and manual live-approval evidence documentation.
